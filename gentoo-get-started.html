Getting started with Gentoo

Installing Gentoo works a bit differently than other Linux distributions.
But don't worry, we got you covered every step of the way.

Step 1: Boot a live environment.

We offer a variety of live environments that allow you to perform the installation.
Choose between a lean Gentoo installation CD or use a LiveCD/DVD of any distribution you like.
Gentoo currently does not offer LiveDVD anymore but it may return in the future.

Step 2: Follow the installation instructions.

The Gentoo Handbook provides detailed documentation that guides you through the installation process.
There is no installation programâ€”you're the installer. That way, you can apply all the customizations you desire.

Ready to go?

These steps are all described in the Gentoo Handbook.
Should you have any questions regarding Gentoo,
or the installation process, we're here for you!
